Find the sum

Select all correct ways to find the sum of integer numbers in the array numbers.

a)

int sum = 0;
for (int i = 0; i < numbers.length; i++) {
    sum += numbers[i];
}
b)
int sum = 0;
for (int i = 1; i < numbers.length; i++) {
    sum += numbers[i];
}

c)

int sum = 0;
for (int number : numbers) {
    sum += number;
}

d)

int sum = 0;
for (int i = 0; i <= numbers.length; i++) {
    sum += numbers[i];
}
